--
Installing the application:

1. Open a command prompt.
2. Set both the parameters in the config file to be true for a clean install, based on both data files
3. Keep the data files in the /data/ directory
4. Run 'python3 install.py'

The database will be installed.


--
Running the application:

1. cd to the 'django/LibrEasy/' directory
2. run 'python3 manage.py runserver'
3. Once server is running, visit 127.0.0.1:8000/app/
4. Refer to the 'QuickStart.pdf' guide for more details.
